Analysis and melody extraction

- If there is an audio signal available, it can be analyzed.
- If the signal has already been analyzed, a melody can be extracted from it.
  During the process of extracting a melody, another one equivalent to its MIDI
  representation is obtained automatically.

These actions are done with the Analysis group in the menu.

• Analyze signal
    Menu: Analysis -> Analyze

• Extract melody
    Menu: Analysis -> Extract melody

   The system will show the user a message once the extraction has been completed.
