MIDI

• Saving a MIDI file
    Menu: File -> MIDI -> Save MIDI...

   The user must indicate the name of the file followed by the extension '.mid'
   Example: midi_out.mid

• Playing MIDI
    Select the MIDI device and instrument from the MIDI group menu


    Select the MIDI play mode from the play radio group.


   Push the 'Play' button to begin the reproduction.
   Push 'Stop' if you wish to finish the reproduction.

* Note: the options in the play radio group are activated depending on the data available.
   If a particular option is deactivated this means that there is no data of that type
   available to be reproduced.
