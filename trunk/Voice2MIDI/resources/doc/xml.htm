XML

The application allows the user to save a melody extracted using the segmentation
process, an edited version of the melody, or its MIDI equivalent in an XML file.



• Savin a melody
    Menu: File -> Melody -> Save melody...

• Saving a MIDI melody
    Menu: File -> Melody -> Save MIDI melody...

In both cases the user must indicate the name of the file followed by the extension '.xml'
Example: melody_out.xml
