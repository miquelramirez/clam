Types of Display

The Voice2MIDI conversion system offers the user several display types for viewing
the data generated by the application.
Click on the following links to find out more about the types of display available.
These displays are accessed through the View group in the menu.

1. Original audio view
2. Analysis view
3. Segmentation view
4. Piano Roll view

* Note: The options in the View group of the menu are activated according to
   data availability. If a particular option is deactivated this means that there
   is no data of that type available to be displayed.
